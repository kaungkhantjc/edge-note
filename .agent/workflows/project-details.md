name: Edge Note
description: I am building a serverless Note Taking app using React Router v7 that can be deployed on Cloudflare Workers.
Framework: React Router v7 (configured for Cloudflare)
Database: Cloudflare D1
ORM: Drizzle ORM (Use drizzle-orm and drizzle-kit)
Styling: Tailwind CSS

Initialized project using:
> npm create cloudflare@latest -- edge-note --framework=react-router
